POSIS

POSIS is a mobile-based Point of Sale and Inventory System built using Android Studio with pure Java and SQLite database. It allows users to efficiently add or delete products, monitor stock levels, and track sales in real time. Designed for simplicity and reliability, POSIS is an ideal solution for small businesses seeking a lightweight and effective inventory management tool.
